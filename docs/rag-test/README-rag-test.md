# Artefatos de teste RAG

Conjunto sintetico para testar ingestao, chunking, classificacao e recuperacao.

- `contrato-desenvolvimento-sistemas-rag.docx`: contrato extenso com clausulas, tabelas e imagem de timeline.
- `calculos-contrato-rag.xlsx`: planilha com abas de premissas, desembolso, custos, SLA e resumo com formulas.
- `arquitetura-sistemas-rag-contractos.docx`: documento tecnico com diagrama de arquitetura, tabelas e requisitos nao funcionais.
- `arquitetura-rag-contractos.png` e `marcos-contratuais.png`: imagens tambem incorporadas nos DOCX para testar OCR/vision isoladamente.
