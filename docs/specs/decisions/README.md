# Decisões (ADRs)

Registre aqui decisões estruturais da POC. Um arquivo por decisão.

Cada ADR contém: contexto, decisão, impacto, alternativas rejeitadas, data e status.

Gatilhos de ADR:
- mudança de fronteira entre camadas
- troca de modelo/dimensão/métrica de embeddings (implica reindexação)
- mudança de schema estrutural (documento, chunk, job)
- mudança na estratégia de chunking que afete retrieval
